This Github repository provides all R Scripts to reproduce the results from the article by J Bretones Santamarina et al. ( https://www.biorxiv.org/content/10.1101/2024.07.16.603530v1 ).

The folder "1_Reactome_processing_and_pruning" contains the code for processing and pruning the original Reactome pathway database.

The folder "2_Combined_enrichment_method" contains R scripts for performing pathway enrichment analysis using the optimized pipeline.

The folder "3_SLi_algorithm" contains R scripts for running the SLi algorithm based on DepMap.
